# Netflix Data Analysis Project

## Project Overview
This project performs Exploratory Data Analysis (EDA) on Netflix dataset using Python.

The goal is to analyze content distribution, trends, ratings, genres, and country-wise contributions.

## Dataset
- 8807 Netflix titles
- 12 columns including type, director, cast, country, rating, duration

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Key Analysis Performed
- Data Cleaning and Handling Missing Values
- Movies vs TV Shows comparison
- Year-wise content growth analysis
- Top contributing countries
- Rating distribution analysis
- Top genres analysis
- Movie duration distribution

## Key Insights
- Movies dominate Netflix content over TV Shows.
- United States and India contribute highest number of titles.
